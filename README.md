
\# Eye Opacity Detector



Программа для анализа помутнения роговицы глаза.



\## Возможности



\- сегментация помутнений (U-Net)

\- анализ изображения

\- сравнение состояния до и после операции

\- визуализация результата



\## Используемые технологии



Python

PyTorch

OpenCV

Tkinte
